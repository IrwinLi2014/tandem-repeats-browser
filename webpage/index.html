Tandem Repeats Browser

usage:
  • File uploaded must be a DNA sequence in FASTA format.
    example:
    >myseq
    ATTT AGTC AGTC AGTC AG

  • Use "Maximum Mismatch Tolerance" parameter to set the maximum percentage of mismatch/insersion/deletion allowed (for each pattern).

  • Use "Window Size" parameter to set the sliding window's size.

  • Use "Alphabet" parameter to set the searched sequence's alphabet.
